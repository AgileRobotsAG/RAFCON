RAFCON -- RMC advanced flow control
===================================

RAFCON (RMC Advanced Flow Control) is an effort of DLR RMC to create a new flow control tool, based on existing ones.
The target of this software is to combine the advantages and features of flow control tools like Bubbles or SMACH
without inheriting their drawbacks.
