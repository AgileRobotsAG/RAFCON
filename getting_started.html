Requirements

The requirements to install and run RAFCON are low:

  • Linux (tested on Ubuntu, SLED and OpenSuse)
  • Python 2.6 or 2.7 (2.7 is recommended)
  • pygtk must be pre-installed (e.g. using apt-get)

Install using Python pip

The easiest way to install RAFCON is using pip. Just enter the following to lines into your favorite shell:

  • pip install rafcon
  • rafcon_start_gui

Install from repository

If you cannot wait for the next release and want to use the latest commits, you can also clone the repository and start RAFCON from there:

  • cd ~/opt_or_so/
  • git clone https://github.com/DLR-RM/RAFCON.git
  • or if you prefer SSH: git clone git@github.com:DLR-RM/RAFCON.git
  • cd RAFCON
  • python setup.py install
  • rafcon_start_gui

This also allows you to run the tests: python setup.py test

How to continue

In the documentation you can read about the basic concepts of RAFCON, explaining states, outcomes, transitions, data ports, data flows and more. There is also a number of step-by-step tutorials, which we recommend to read.
