Requirements

These are the requirements to your system that net to be fulfilled before installing RAFCON (use for example apt-get to install those packages):

  • Linux (tested on Ubuntu, SLED and OpenSuse)
  • Python 2.7
  • pygtk
  • python-setuptools
  • pip
  • python-opengl
  • python-gtkglext1
  • python-gtksourceview2
  • pylint

Install using Python pip

The easiest way to install RAFCON is using pip. Just enter the following to lines into your favorite shell:

pip install rafcon

Start RAFCON

Next, all you have to do for starting RAFCON is calling rafcon from the shell:

rafcon

Detailed instructions

More details about how to install RAFCON can be found in the README.
